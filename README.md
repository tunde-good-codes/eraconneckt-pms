# EraConneckt PMS Landing Page

Welcome to the landing page for **EraConneckt PMS**, a professional Property Management System. This site was designed to showcase the key features, benefits, and value of the EraConneckt platform and to serve as a marketing entry point for potential clients, investors, and users.

##  Key Features

- Modern and responsive design for desktop and mobile  
- Hero section to highlight value proposition  
- Features overview (what EraConneckt PMS offers)  
- Screenshots or UI preview section  
- Call-to-action (CTA) for signup or demo request  
- Contact form or CTA to get in touch  

##  Tech Stack

- Frontend: HTML, CSS, JavaScript , ReactJs
- Deployment: Hosted on **Render** (https://eraconneckt-pms.onrender.com)  
- Optional backend integrations for PMS product

## How to Run Locally

1. Clone the repository  
2. Install dependencies  

## Purpose & Goals

This landing page is intended to:
- Introduce **EraConneckt PMS** to prospective customers  
- Explain core features and benefits of the system  
- Capture leads via contact form or demo request  
- Serve as a polished, professional front-facing web presence for the product
